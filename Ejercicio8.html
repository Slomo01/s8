Lista de Lenguajes de Programacion populares

Java
Java es un lenguaje de programación y una plataforma informática que fue
comercializada por primera vez en 1995 por Sun Microsystems.​​
Python
es un lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia
de datos y el machine learning
C++
es un lenguaje compilado que fue desarrollado en 1980 por Bjarne Stroustroup en los laboratorios At&T
como una extensión orientada a objetos del lenguaje C
Ruby
Es un lenguaje de programación dinámico y de código abierto enfocado en la simplicidad y productividad.
