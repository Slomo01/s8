Ejemplo de Lista de Descripcion

TCP
El Protocolo de transmision es un metodo(protocolo)utilizado junto con el protocolo de internet(IP)para enviar datos en forma de mensaje,llamadas paquetes entre ordenadores atraves de la red de Internet.
IP
El protocolo de internet es el moetodo protocolo mediante el cual se envian datos de un ordenador a otro en internet.Cada ordenador en internet esta identificado de forma unica por una direccion de IP.
FTP
El protocolo de transferencia de archivos es un protocolo utilizado para intercambiar archivos entre ordenadores de Internet.
HTTP
El protocolo de tranferencia de hipertextoes el protocolo utilizado para intercambiar texto, imagenes graficas, sonido, video y otros archivos multimedia en la Web.
